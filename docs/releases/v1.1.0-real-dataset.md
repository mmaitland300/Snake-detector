# v1.1.0-real-dataset Release Package

This release packages the current real-photo Snake Detector demo model used by the live Hugging Face Space.

## Status

- Intended GitHub tag: `v1.1.0-real-dataset`
- Live app: [mmaitland-snake-detector-demo.hf.space](https://mmaitland-snake-detector-demo.hf.space)
- Space repo: [huggingface.co/spaces/mmaitland/snake-detector-demo](https://huggingface.co/spaces/mmaitland/snake-detector-demo)
- Model type: Keras model trained with InceptionV3 preprocessing
- Task: binary `snake` vs `no_snake` image classification
- Safety boundary: demo only; not species identification or field-ready wildlife handling guidance

## Dataset Summary

The training/evaluation corpus was collected with the iNaturalist workflow documented in [real_image_collection.md](../real_image_collection.md) and [collection_runs.md](../collection_runs.md).

Manifest summary:

| Field | Value |
| --- | ---: |
| Manifest rows | 6321 |
| Provider | iNaturalist |
| `snake` rows | 1271 |
| `no_snake` rows | 5050 |
| Photo licenses included | CC0, CC-BY, CC-BY-SA |

Split summary:

| Split | `snake` | `no_snake` |
| --- | ---: | ---: |
| Training | 901 | 3605 |
| Validation | 115 | 417 |
| Testing | 255 | 1028 |

Raw image files and bulk split directories remain outside git. The release publishes the trained model and evaluation artifacts, not the full image corpus.

## Claim Boundary

Good public claim:

> The live Hugging Face demo uses a real-photo iNaturalist-trained Keras model, with held-out test metrics and release assets documented in `v1.1.0-real-dataset`.

Avoid:

- Claiming species identification
- Claiming field-ready wildlife safety or handling guidance
- Comparing placeholder metrics directly against real-photo metrics as if they were the same dataset
- Publishing raw third-party images in git
